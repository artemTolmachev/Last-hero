# Интернет магазин
* Был создан с помощью библиотеки React
## Содержит:
* Модальные окна
* Корзина товаров
* Счетчик товаров
* Аккордион 
* Табы 
* Анимации 
* Слайдеры
* Range slider 
* Чекбоксы
* Формы обратной связи
## Технологии
* Javascript
* PHP
* SASS
* React
* Router 
* Hooks
* Redux.js
* React-spring (animation library)
* React-slick (corouel)

[DEMO](https://last-hero-react1.vercel.app/)
